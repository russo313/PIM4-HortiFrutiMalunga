Sobre a implementação

storage Back-end em .NET + PostgreSQL

A API está rodando em containers Docker (imagem .NET 8 com Entity Framework Core e banco PostgreSQL 16). Todas as páginas do painel consomem os endpoints reais (/api/categories, /api/products, etc.), e os dados exibidos foram semeados diretamente no banco para demonstrar as funcionalidades.

Para testar operações protegidas (cadastro de produtos, vendas, movimentações manuais) basta autenticar na tela de Login. O token JWT é enviado em todas as requisições subsequentes.

dashboard Painel e navegação
  • O dashboard inicial apresenta indicadores calculados em tempo real a partir da API.
  • Categorias, produtos, clientes, vendas, movimentos e alertas mostram exatamente o que está no banco.
  • O proxy (proxy.conf.json) encaminha as chamadas do Angular para http://localhost:8080/api.
