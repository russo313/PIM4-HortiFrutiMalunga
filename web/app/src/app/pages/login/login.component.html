Sobre esta demonstração

info Projeto estático para apresentação

Este ambiente foi preparado apenas para exibição visual no Vercel. Todas as telas exibem dados fictícios simulando o funcionamento do hortifruti Malunga.

O fluxo de autenticação original foi desativado, portanto nenhuma credencial é necessária. Utilize o menu para navegar entre as páginas e visualizar as simulações de categorias, produtos, clientes, vendas e alertas.

Caso deseje reaproveitar este layout com uma API real, basta reconectar as chamadas HTTP no código fonte.
